## ⚡ Install on Kali Linux — Copy Paste These Exact Commands

```bash
cd ~/Downloads
unzip MobilNumTrack.zip
cd MobilNumTrack
pip3 install -r requirements.txt --break-system-packages
python3 MobilNumTrack.py
```

**That is everything. No git. No username. No password. No token.**
